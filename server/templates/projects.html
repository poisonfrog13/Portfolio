  1. Habit Tracker

Habit Tracker App

The Habit Tracker app is a versatile tool designed to help users effortlessly monitor and manage their habits. Accessible on any device with internet, fostering positive habits effortlessly.

About Habit Tracker

Motivation

The concept for the Habit Tracker app emerged from a collaboration with a frontend developer friend. The inspiration came from my desire to track how many hours I spend programming and how many cups of coffee he drinks during one working day as a developer.

Source Code

The backend built with Django and DRF supports fundamental CRUD operations for habits and daily records, encompassing both list and detail views.

The app built features a REST API providing JSON data and is intended to be consumed by a front end web application.

Stateless token-based authentication system ensures secure user access and interactions.

Proficient software testing, including pytest and pytest-django, validates the app's reliability and API functionality.
